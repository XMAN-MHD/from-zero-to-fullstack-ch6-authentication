# 🚀 From Zero To Full-Stack 

> From beginner to professional Full-Stack Software Engineer by building real-world applications.

This repository is not just another tutorial project.

It is my personal engineering journey where I progressively build a production-ready full-stack application while learning modern software engineering practices used by professional development teams.

Instead of learning technologies in isolation, every concept is applied to a real application.

# ⚙️ Running Locally

Clone the repository

```bash
git clone https://github.com/XMAN-MHD/from-zero-to-fullstack-ch5-devops.git
```

Install dependencies

```bash
npm install
cd backend
npm install
```

Run Docker

```bash
docker compose up --build
```

Frontend

```
http://localhost:3000
```

Backend

```
http://localhost:3001
```
